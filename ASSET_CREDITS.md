# Asset credits

All procedural meshes authored by the Aetherwake project remain part of the
project source. The following third-party source material is bundled under CC0:

- **Leaves Forest Ground** by Dario Barresi (processing) and Dimitrios Savva
  (photography), downloaded from Poly Haven. Diffuse, OpenGL normal, and packed
  AO/roughness/metalness maps are included at 2K resolution.
  https://polyhaven.com/a/leaves_forest_ground

- **Rock Moss Set 01** by Kless Gyzen, downloaded from Poly Haven. Six scanned
  rock meshes are bundled as individually instanced 1K runtime GLBs.
  https://polyhaven.com/a/rock_moss_set_01

- **Tree Stump 02** by Rob Tuytel, downloaded from Poly Haven. The 1K scanned
  mesh is bundled as an optimized runtime GLB.
  https://polyhaven.com/a/tree_stump_02

- **Fern 02** and **Dead Tree Trunk**, downloaded from Poly Haven's Pine Forest
  collection. Their 1K meshes are bundled as optimized runtime GLBs.
  https://polyhaven.com/a/fern_02
  https://polyhaven.com/a/dead_tree_trunk

- **Fir Tree 01** by Rico Cilliers, Rob Tuytel, and Dimitrios Savva. The
  lightest official tree variant is range-extracted, decimated to a controlled
  hero-tree budget, and bundled locally.
  https://polyhaven.com/a/fir_tree_01

Poly Haven assets are released under the Creative Commons CC0 1.0 Universal
license: https://polyhaven.com/license
